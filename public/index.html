UFSA Concursos API

Welcome to the UFSA Concursos API. This API provides access to public tender information from UFSA.

Available Endpoints

  • /api/concursos/abertos - Get all open tenders
  • /api/concursos/adjudicados - Get all awarded tenders
  • /api/concursos/ajustes-directos - Get all direct adjustments
  • /api/concursos/provincias - List all provinces
  • /api/concursos/tipos - List all tender types
  • /api/concursos/entidades - List all contracting entities
  • /api/status - Check API status

For detailed documentation, please refer to the API documentation.
